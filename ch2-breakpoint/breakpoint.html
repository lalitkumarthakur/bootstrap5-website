Ch2 - Breakpoint

Breakpoints are used to define the "width covered by that specific breakpoint of that specific class" Example - If we need to make of website specific to Tablet device, than we will use "md class". This/ and other class will define specific widths required for all the screen sizes. Based on the screen size we require to build the wesbsite, we use the required container class for the screen size required for that situation/client requirment.

X-small devide has no class. it's range is 0 to 575px

Small devices uses sm calss. Range - 576px to 767px

Medium devices uses md calss. Range - 768px to 991px

Large devices uses lg calss. Range - 922px to 1198px

Extra Large devices uses xl calss. Range - 1200px to 1139px

Extra extra devices uses xxl calss. Range - greter than 1400px
