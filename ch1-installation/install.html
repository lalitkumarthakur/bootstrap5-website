Welcome to bootstrap Installation !


In this chapter, we will learn how to install/include bootstrap in your HTML file in order to use it.

Here are the steps to include bootstrap 5.0.2 in your HTML file -

  • Please visit https://getbootstrap.com/docs/5.0/getting-started/introduction/ this link and copy the CSS file Link.
  • Include the CSS file link inside the HEAD TAG.
  • Go back to the browser and copy the javascript file link.
  • Past the javascript file link at the end and above the closing of the body tag.
  • Congratualtion! You have successfully included the Bootstrap file links into the HTML file and you are ready to use Bootstrap 5.0.2
