Ch3 - Container Class

Container Class are used to define the width of the container. Based on the screen size we require to build the wesbsite, we use the required container class for the screen size required for that situation/client requirment.

This container class named "container" will take 100% width less than 576px. After corssing 575px, it will be in center and adjust itself according to screen size.

This container class named "container-sm" will take 100% width within 576px to 767px. After corssing that, it will be in center and adjust itself according to screen size.

This container class named "container-md" will take 100% width within 768px to 991px. After corssing that, it will be in center and adjust itself according to screen size.

This container class named "container-lg" will take 100% width within 922px to 1198px. After corssing that, it will be in center and adjust itself according to screen size.

This container class named "container-xl" will take 100% width within 1200px to 1139px. After corssing that, it will be in center and adjust itself according to screen size.
